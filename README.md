# Algorithm Resources

## How to Get Started
1. Clone the repo:
    ```git clone https://github.com/joellepfernandez/algorithm_resources.git```
2. Install dependencies:
    ```npm install```

## How to Run Tests
To run all tests:
```npm test```

To run a specific test:
```npm test TESTNAME```
